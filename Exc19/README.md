# ABAP CDS Fortgeschrittene Konzepte

**Metadaten Erweiterung**<br>
<br>
Erstelle zum View ZI_Umsatz_Kundeverhalten[xx] eine Metadatenerweiterung ZI_Umsatz_Kundenverhalten_UI[xx]<br>
<br>
Alle UI Annotationen für Oberflächen werden in dieser separaten Metadatenweiterung gepflegt<br>
<br>
Pflege für jedes Feld einen Label (LineItem)<br>
Füge die Felder nicht einzeln hinzu, sonder per Code-Completion<br>
<br>
Die Metadaten sind im Layer Partner<br>
Stelle den Ursprungsview so um, dass er eine Erweiterung erlaubt<br>
Prüfe die aktiven Annotationen von ZI_Umsatz_Kundeverhalten[xx] <br>
