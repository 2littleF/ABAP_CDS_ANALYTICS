# ABAP CDS Fortgeschrittene Konzepte

**Erweiterung eines CDS View**<br>
<br>
Erstelle die Erweiterung ZI_FLUG_ROUT_KOMPL[xx] basieren auf dem View ZI_FLUGPLAN[xx]<br>
<br>
Führe zusätzliche Assoziationen auf ZI_Airport[xx] ein, jeweils für Abflug- und Zielflughafen<br>
Exponiere die beiden Assoziationen in der Feldliste<br>
<br>
<br>
Prüfe den View ZI_FLUGPLAN[xx] im Data Preview und folge dem Pfad zu den Klartextnamen der Flughäfen<br>
